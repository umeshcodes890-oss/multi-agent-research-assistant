# Multi-Agent Research Assistant

A Multi-Agent Research Assistant built using Python, LangGraph, Ollama, Gemma 3 4B, and Streamlit.

## Features

* Planner Agent
* Search Agent
* Research Agent
* Writer Agent
* Critic Agent
* Fact Checker Agent
* Supervisor Agent

## Tech Stack

* Python
* LangGraph
* Ollama
* Gemma 3 4B
* Streamlit

## Project Status

Current Completion: 75%

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
